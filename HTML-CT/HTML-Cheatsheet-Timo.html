Cheatsheet für HTML

HTML

  • HTML steht für HyperText Markup Language
  • Die grundlegende Syntax eines HTML Befehls ist: <ELEMENT>INHALT</BEFEHL>, es gibt jedoch auch Ausnahmen

Struktur einer HTML Datei

Allgemeines zum Dokument

  • Der Befehl <!DOCTYPE html> ist sehr wichtig und muss in der ersten Zeile des Dokuments stehen.
    Dies zeigt dem Browser an, dass es sich um eine HTML Datei handelt, DOCTYPE ==> Document Type
  • Kommentare: <!--So wird ein Kommentar in einem HTML Dokument geschrieben-->
  • Leerzeichen/Leerfläche: Der Browser ignoriert Leerzeichen oder Absätze, sie sind aber extrem Hilfreich, um Ordnung in einem HTML Dokument zu behalten

Head und Body

  • Es gibt den head und den body tag
  • Der head tag beherbergt Dinge wie den Titel, Stylesheets oder das das charset (bsp. UTF-8)
  • Der body tag beinhaltet alle sichtbaren Inhalte einer Website

Überschriften

  • Es gibt verschieden große Überschriften in HTML. Diese werden von h1 bis h6 benannt, wobei h1 die größte und h6 die kleinste Überschrift ist

Text Blöcke

Es gibt verschiedene Befehle, die etwas mit Text zu tun haben:
  • <p>: Englisch: Paragraph
  • <span>: Dazu gedacht, um mit Hilfe von CSS einzelne Wörter oder wie hier Buchstaben zu bearbeiten
  • <div>: Englisch: HTML Document Division Element, allgemeine Umgebung für fließenden Text

Wort- und Text hervorhebung

  • <em> und <strong> sind dafür da, um Text hervorzuheben
  • Mit dem Befehl <br> kann man einen Zeilenumbruch in seinen Text einbauen

Verlinkungen

  • Mit <a>, dem "anchor" tag, erstellt man auf der Website einen Link zu z.B. einer anderen Seite
  • Die richtige Syntax lautet hier: <a href="Webadresse der zu verlinkenden Seite">Dargestellter Text für den Link</a>

Listen

Es gibt zwei Arten von Listen:
  • Einmal die ungeordnete Liste (<ul>), die ich auch schon die ganze Zeit auf dieser Seite verwende
  • Und die geordnete Liste (<ol>), dies ist eine nummerierte Liste

Bilder und Videos

Mit HTML kann man in eine Website relativ leicht Bilder und Videos einbinden:
  • Bilder lassen sich mit dem Befehl <img src="Bildquelle"> einbinden.
    Dabei benötigt man keine closing tags
  • Videos lassen sich mit der <Video> Umgebung und dem <source src="Videoquelle" type="z.B. video/ogg"> Befehl ebenfalls einbinden. Hier kann man noch mit dem Attribut "controls" Video-Kontrollelemente wie z.B. Pause einfügen
  • Sogenannte Gifs (sich bewegende Bilder) kann man mit dem Befehl <embed> in seine Seite einbinden
  • Sowohl bei Bildern, Gifs als auch bei Videos kann man mit dem Attribut "alt" einen Alternativtext anzeigen lassen falls das Bild/Video nicht richtig gelden wurde

Tabellen

Befehl Erklärung des Befehls
<table> Erzeugt eine Tabelle
<th> Erzeugt eine Überschrift => Table headline
<thead> Erzeugt die "table head" Umgebung, die ähnlich zur HTML-head Umgebung ist.
<tbody> Erzeugt die "table body" Umgebung, die ähnlich zur HTML-body Umgebung ist.
<tr> Erzeugt eine Reihe in der Tabelle => Table row
<td> Tabelleneintrag => Table data
<tfoot> Letzte Zeile in der Tabelle => Table foot

HTML Semantik

<header> (Überschriften), <nav> (Navigationslinks), <main> (Hauptelemente) und <footer> ("Fußzeile") sind für die allgemeine Struktur von HTML-Dokumenten zuständig.

<section> ist dafür da, um Dinge wie Textabschnitte, Überschriften oder andere Umgebungselemente zu umschließen.

<article> enthält Inhalte wie Artikel, Blogs, Kommentare usw.

<figure> umschließt alle Elemente einer Seite, die Medien wie Bilder oder Videos enthalten.
Dabei ist <figcaption> dafür da, um die Medien in <figure> zu beschreiben.

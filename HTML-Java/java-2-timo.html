Java Zusammenfassung, Buch 5.3.1 - 5.3.4

5.3.1 Erste Schritte

Hello World

Traditionell schreibt man in einer neuen Programmiersprache erstmal ein sogenanntes "Hello World" Programm. Dieses ist ein ganz simples Textausgabeprogramm in der Konsole. Eine Konsolenausgabe wird in Java über den Befehl "System.out.println()" realisiert.

5.3.2 Anweisungen

  • Anweisungen müssen immer mit einem Semikolon (;) beendet werden
  • In Struktogrammen werden Anweisungen jedoch als Rechteck dargestellt

5.3.3 Variablen

Einfache Datentypen

Es gibt verschiedene Datentypen in der Java Programmiersprache:
  • string (Text)
  • char (Buchstabe)
  • Ganzzahlen
    • int
    • short
    • byte
    • long
  • float (Fließkommazahl)
  • boolean (Wahr oder Falsch)

Variablen deklarieren

  • Variablen deklariert man mit folgender Syntax: [Datentyp] [Name] = [Wert]
